# Trans Star - Transport & Logistics Business Website

A modern, responsive website for Trans Star, a cold-chain transport and logistics company serving New Zealand.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design matching the PDF specifications
- **Interactive Quote Form**: User-friendly form with validation
- **Smooth Scrolling**: Enhanced navigation experience
- **Temperature Monitoring Display**: Visual representation of cold-chain monitoring

## Sections

1. **Header**: Sticky navigation with contact information and quote button
2. **Hero Section**: Main value proposition with call-to-action buttons
3. **Trusted By**: Social proof section
4. **Problem/Solution**: Highlights the value proposition
5. **Who We Work With**: Target customer segments
6. **How It Works**: Three-step process explanation
7. **Quote Form**: Contact form for requesting quotes
8. **Footer**: Contact information, services, and certifications

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript

## Getting Started

Simply open `index.html` in a web browser to view the website.

## Customization

- Colors can be adjusted in the `:root` CSS variables in `styles.css`
- Form submission handling can be configured in `script.js` (currently logs to console)
- Content can be modified directly in `index.html`
